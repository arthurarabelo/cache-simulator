Para executar o programa, basta rodar o seguinte comando em um terminal:

python simulador.py <tamanho_da_cache> <tamanho_da_linha> <tamanho_do_grupo> <arquivo_de_entrada>

Obs: no exemplo 5 dos exemplos disponibilizados, os endereços deveriam pertencer os dois ao conjunto 0, visto que os dois números são pares
e um número par mod 2 só pode ser igual a 0.
